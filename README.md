# 🏙️ Municipalités Chaudière-Appalaches  
Application Tkinter affichant les municipalités du Québec avec cartes interactives Folium

---

## 📌 Description

Cette application Python permet d’explorer les municipalités de la région **Chaudière-Appalaches** à travers une interface graphique moderne construite avec **Tkinter**.  
Elle inclut :

- Une liste des MRC  
- Une liste des municipalités associées  
- Un panneau de détails complet  
- Des boutons arrondis personnalisés (Canvas)  
- L’affichage des municipalités sur une **carte interactive Folium**  
- L’affichage de **toutes** les municipalités d’un coup

Ce projet combine **Tkinter**, **Pandas**, **Folium** et **PIL** pour offrir une interface simple, intuitive et visuellement agréable.

---

## 🚀 Fonctionnalités

### ✔ Sélection des MRC  
Affiche automatiquement les municipalités associées.

### ✔ Sélection d’une municipalité  
Montre les informations détaillées :

- Nom  
- Population  
- Latitude / Longitude  
- Informations supplémentaires (maire, téléphone, site web, logo si disponible)

### ✔ Carte Folium interactive  
- Affichage d’une municipalité spécifique  
- Affichage de toutes les municipalités  
- Couleurs différentes selon la MRC  
- Zoom automatique sur la municipalité sélectionnée

### ✔ Interface moderne  
- Boutons arrondis dessinés en Canvas  
- Thème sombre  
- Panneau de détails avec padding  
- Organisation claire en colonnes
